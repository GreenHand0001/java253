# 冬奥会科普管理系统 java253

## 项目概述

冬奥会科普管理系统是一个基于前后端分离架构的信息管理系统，旨在为用户提供全面、专业的冬奥会相关知识管理。系统采用 SpringBoot 和 Vue.js 作为主要的技术栈，确保了系统的稳定性和用户界面的流畅性。

## 技术栈

- 后端：SpringBoot
- 前端：Vue.js
- 数据库：MySQL
- 开发工具：Idea，vscode
- 其他：mybatis-plus，element

## 功能模块

### 系统角色

- 普通用户
- 超级管理员

### 功能列表

- 项目类型管理
- 冬奥项目管理
- 精彩视频管理
- 冬奥论坛管理
- 轮播图管理
- 公告管理
- 用户管理
- 个人中心
- 个人后台

## 运行环境

- 后端：Java 1.8 或以上版本
- 前端：现代浏览器支持
- 数据库：MySQL 5.7 或以上版本
